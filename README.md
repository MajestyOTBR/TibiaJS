# TibiaJS

### Requirements
* Node 8+


### How to run
* `git clone https://github.com/RemoveIt/TibiaJS.git`
* `cd TibiaJS`
* `git submodule init`
* `git submodule update`
* `npm install`
* `npm run build`
* `npm run run` or `cd Out` & `node Server.js`

### How to update cloned repo
* `git pull origin master`
* `git submodule foreach git pull origin master`
